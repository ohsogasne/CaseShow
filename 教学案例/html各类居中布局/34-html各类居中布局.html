行内元素——display：inline

1.不会自动换行，无法设定宽度与高度（可设定line-height指定高度，宽度随内容变化），margin水平有效垂直无效，padding水平有效垂直有特殊效果[不是边距效果]，具体表现是影响视觉不影响位置

2.常见的行内元素包括：

元素名称 说明
a 锚点
b 粗体(不推荐)
span 常用内联容器，定义文本内区块
img 图片
input (输入框)
select 项目选择
strong 粗体强调
label 表格标签
cite 引用
textarea 多行文本输入框
em 强调
br 换行
sup\sub\u\tt\strike\q\em\ 比较少用需要再自己去查

块级元素——display：block

1.默认特点： 独占一行，使用多个块级标签时自动换行。 |可以设置width,height，margin,padding等属性 |宽度默认值为 100%

2.常见的块级元素包括：

元素名称 说明
div 常用
h* 各类标题
ol 有序表单
dl 定义列表
ul 无序表格
form form交互表单
hr 分隔线
p 段落
table 表格

行内块元素——display：inline-block

1.默认特点：元素排在一行，会有空白缝隙 |可以设置width,height，margin,padding等属性 |默认宽度由随内容变化

2.常见的行内块级元素包括：

元素名称 说明
button 按钮
input 输入框
img 图片
iframe 框架
textarea 多行文本框
select 选项
